# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-18

### Added

- Multi-tab PTY terminal console for running Claude Code sessions (Beta)
- Session analytics with cost tracking, token usage breakdowns, and tool frequency analysis
- Block-based usage monitoring with weekly and monthly views
- Visual workflow canvas builder for multi-step agent orchestrations
- AI agent builder with configurable models and tools
- MCP server management for adding, configuring, and monitoring servers
- Settings and configuration hub for Claude Code integration
- Skill library for browsing, editing, and organizing reusable prompts
- Plugin marketplace with GitHub search for discovering packages (Beta)
- AI-powered Git hook configuration generator (Beta)
- JSONL session log parser for Claude Code output
- SQLite-based local storage with auto-creation on first run
- WebSocket server for real-time terminal communication
- Dark mode support
